# 🎯 SUPER SIMPLE GUIDE (5 Minutes)

## What This Bot Does
Copies messages from one Telegram channel to another, even if forwarding is blocked.

---

## SETUP (Do Once)

### 1. Start the Bot
```
Go to Telegram
Search: @goclassbckup_bot
Tap: START
```

### 2. Add Your Bot Account
```
Send: /settings
Tap: 🤖 My Bots
Tap: ✚ Add bot
Send: Your bot token (get from @BotFather)
```

### 3. Add Target Channel (Where messages go)
```
Send: /settings
Tap: 📨 My Channels
Tap: ✚ Add Channel
Send: t.me/yourchannelname
(or forward a message from that channel)
```

### 4. Make Bot Admin
```
Go to your target channel
Add bot as member
Make it ADMIN
Give permissions: Send, Edit, Delete messages
```

**Done! Setup complete.** ✅

---

## USE BOT (Do Every Time)

### 5. Start Forward
```
Send: /fwd
```

### 6. Pick Source (Where messages come from)
```
Choose ONE:
a) Send: t.me/sourcechannel/12345 (message link)
b) Forward: A message from source to bot
c) Send: me (if from Saved Messages)
```

### 7. Pick Skip Number
```
Send: 0 (forward all)
or
Send: 5 (skip first 5, forward rest)
```

### 8. Confirm
```
Check the information
Tap: YES ✅
```

### 9. Wait
```
Bot shows progress
When done: 🎉 FORWARDING COMPLETED
```

**Finished!** 🚀

---

## That's It!

Just repeat steps 5-9 whenever you want to forward.

## Common Problems & Quick Fixes

| Problem | Fix |
|---------|-----|
| Bot can't send messages | Make bot ADMIN in target channel |
| "Can't access source" | Make bot ADMIN in source channel |
| "Invalid link" | Use: t.me/channel/123 format |
| Bot not responding | Send: /start |
