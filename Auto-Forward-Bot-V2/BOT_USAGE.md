## 🤖 Bot Usage Instructions

### IMPORTANT: How to Use This Bot

The bot operates as **@goclassbckup_bot**

#### Step 1: Open Private Chat
- Search for **@goclassbckup_bot** on Telegram
- Open the private chat (NOT a channel, NOT a group)
- This is where you send commands to the bot

#### Step 2: Send /start Command  
- In the private chat with @goclassbckup_bot
- Type: `/start`
- The bot will respond with a menu

#### Step 3: Select Source Channel
- Click "📥 Select Source Channel"
- Send the channel URL you want to forward from
- Examples:
  - `https://t.me/+3CoEtU8yo0hhZGVh`
  - `https://t.me/c/3895961049/`
  - `@channelname`

#### Step 4: Select Topic
- The bot will show available topics
- Click the topic you want to forward
- Bot will start forwarding all messages to @gatearshadbackup

### Important Notes:
⚠️ **PRIVATE CHAT ONLY**: You MUST send commands in a direct private message to @goclassbckup_bot
❌ Do NOT send commands in the @gatearshadbackup channel
❌ Do NOT send commands in groups

### Commands:
- `/start` - Show main menu
- Send channel URL - Connect to source channel
- Click topic button - Start forwarding

### Troubleshooting:
If bot doesn't respond:
1. Make sure you're in a PRIVATE chat with @goclassbckup_bot
2. Try sending `/start` again
3. Wait 2-3 seconds for response
4. Check that Render deployment is showing "Live" status
